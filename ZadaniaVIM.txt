Zadanie 4 - Podaj nazwę pliku konfiguracyjnego nano i treść, która umożliwi, aby tabulatory były automatycznie zamieniane na spacje (za każdym uruchomieniem):
Plik konfiguracyjny nano - ~./nanorc

Druga część zadania -
set tabstospaces
set tabsize 4 

Zadanie 5 - Podaj nazwę pliku konfiguracyjnego vim i treść, która umożliwi, aby tabulatory były automatycznie zamieniane na spacje (za każdym uruchomieniem).
Plik konfiguracyjny vim - ~/.vimrc

Druga częśc zadania:
 
set expandtab - Zamienia tab na spacje 
set tabstop=4 - Szerokość tabulatora 
set shiftwidth=4 - Liczba spacja dla wcięia
set softtabstop=4 - Liczba spacji po nacisnieciu klawisza tab
